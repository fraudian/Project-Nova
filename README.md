# Project Nova Launcher 🖥️

Project Nova Launcher is a legacy version of the Project Nova launcher.  
This repository includes Source Code from an older Build that has been restored and fixed to bypass previous Login issues.

> ⚠️ Note: This is an Old Build and is not up to date with the current Nova Launcher.

## Features 🎮

- Fixed Login Screen issue from Original Build
- Legacy Launcher functionality restored
---

## Requirements 🛠️

Before running this project, make sure you have:

- Visual Studio 2022 (Community / Professional / Enterprise)
- .NET Desktop Development workload installed
- .NET Framework (usually 4.7.2 or 4.8 depending on project)
- Git (optional)

---

## How to Download ✅

### Option 1: Download ZIP
1. Click the green **Code** button on **GitHub**  
2. Select **Download ZIP**  
3. Extract the folder anywhere on your PC  
4. Open **NovaLauncher.sln** > **Visual Studio 2022**

### Option 2: Clone with Git
```bash
git clone https://github.com/fraudian/Project-Nova.git
```
